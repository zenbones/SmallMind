= SmallMind
:toc: left
:toclevels: 2
:icons: font

SmallMind is a comprehensive multi-module Java framework for building enterprise applications.
It provides a curated set of integrations, abstractions, and utilities designed to work together
coherently across the full stack of a modern Java application — from persistence and messaging
to scheduling, web services, and observability.

The project targets Java 25 and is built with Maven. It integrates with widely-adopted
libraries and platforms including Spring, Hibernate, Jakarta EE, Jetty, Netty, Kafka,
RabbitMQ, MongoDB, and AWS, providing a consistent programming model across all of them.
